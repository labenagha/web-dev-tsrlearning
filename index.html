12 Weeks, Step-by-Step

Week 1: What is a Website?
  • • How the internet works (simplified)
  • • What is HTML and why do we need it?
  • • Setting up your coding environment
  • • Your first "Hello World" webpage
  • • Understanding file structure and folders
Week 2: Text and Structure
  • • Headings and paragraphs
  • • Bold, italic, and text formatting
  • • Creating lists (numbered and bullet points)
  • • Adding line breaks and spacing
  • • Comments in HTML code
Week 3: Links and Images
  • • Adding images to your webpage
  • • Creating links to other pages
  • • Email links and phone links
  • • Making a simple navigation menu
  • • Project: Build your first personal "About Me" page
Week 4: Introduction to CSS
  • • What is CSS and how does it work?
  • • Connecting CSS to your HTML
  • • Changing colors and backgrounds
  • • Font families, sizes, and text styling
  • • Making your first styled webpage
Week 5: Layout and Positioning
  • • Understanding the box model
  • • Padding, margins, and borders
  • • CSS Flexbox for easy layouts
  • • Centering content and alignment
  • • Creating columns and rows
Week 6: Responsive Design
  • • Making websites work on phones and tablets
  • • CSS media queries (mobile-first approach)
  • • Flexible images and text
  • • Testing on different screen sizes
  • • Project: Create a responsive business card website
Week 7: Programming Fundamentals
  • • What is programming? Think like a computer
  • • Variables: storing information
  • • Numbers, text, and true/false values
  • • Your first JavaScript alert box
  • • Console.log for testing your code
Week 8: Making Things Happen
  • • Click events: making buttons work
  • • Changing text and images with JavaScript
  • • Simple if/else decisions
  • • Getting user input from forms
  • • Creating a simple calculator
Week 9: Interactive Features
  • • Form validation (checking user input)
  • • Show and hide content
  • • Simple animations and effects
  • • Image galleries and slideshows
  • • Project: Build an interactive photo gallery
Week 10: Tailwind CSS Bonus
  • • Introduction to utility-first CSS
  • • Quick styling with Tailwind classes
  • • Building components faster
  • • Responsive design made easy
  • • Converting your previous projects to Tailwind
Week 11: Portfolio Website
  • • Planning your personal website
  • • Professional design principles
  • • Showcasing your projects
  • • Contact forms and social links
  • • Making it mobile-friendly
Week 12: Launch Your Website
  • • Choosing a domain name
  • • Free hosting options (GitHub Pages, Netlify)
  • • Publishing your website online
  • • Basic SEO for beginners
  • • Final presentations and celebration!

Frequently Asked Questions

Common questions from our beginner students

Do I really need ZERO experience to join?

Absolutely! We designed this course specifically for complete beginners. We start with "What is a website?" and build from there. Many of our most successful students had never written a line of code before joining us.

What if I'm not "good with computers"?

Perfect! Many of our students felt the same way. If you can use email and browse the internet, you have all the computer skills needed to start. We'll teach you everything else step by step.

Am I too old to learn programming?

Never! Our oldest successful graduate was 67. Programming is a skill like any other - with patience and practice, anyone can learn. We've had students from 18 to 67 succeed in our program.

What equipment do I need?

Just a computer (Windows, Mac, or Chromebook) and internet connection. We'll help you set up all the free software you need in the first class. No expensive equipment required!

What if I fall behind or don't understand something?

No one gets left behind! With only 8 students per class, we give personal attention to everyone. We also offer free review sessions and one-on-one help when needed. We repeat concepts until everyone gets it.

Can I really get a job after 12 weeks?

While we can't guarantee job placement, many of our graduates have found work, started freelancing, or built their own websites. We focus on practical skills and help you build a portfolio that showcases your abilities.
